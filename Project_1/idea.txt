이동?
 - 기본 이동 좌우 방향키
 - 가능하면 벽타기
   - 점프 중 벽 옆부분에 닿은 상태로 E키 누르면 달라붙음 
   - 조금씩 미끄러져 내려감
   - 벽에 마찰력 적용..
   - 미끄러져 내려올 때 점프키 누르면 반대쪽으로 점프
   - 점프 중 다시 벽타기 가능.
 - 점프(이차함수), 이단점프까지
 - 물에서 이동
   - 시간이 지날수록 가라앉음
   - 수중에서 좌우방향키 + 점프키 -> 빠르게 이동
   - 수면에서 위아래방향키 + 점프키 -> Jumpheight 적어진 값으로 약간만 점프
   - 수면에서 헤엄 시 물 튀기는 애니메이션
   - 산소 게이지 적용
      - 전부 소모되면 체력 틱뎀
      - 적어질수록 화면에 빨간 이펙트 => 원래 체력 적을때부터 적용
 - 시점? 캐릭터 위치 따라 뒷배경을 이동.
 - 피격?
   - 넉백 구현

지형? 
 - 탈 수 있는 벽
 - 물
 - 함정
 - 맵 곳곳에 아이템 배치
   - 일회용 폭탄
   - 회복약 (정해진 체력바 안에서 회복할 수 있는 하트)
              (정해진 체력바 이외에 추가되는 하트)
   - 영구적으로 공격력 올라가는 아이템
 - 추가?
   - 현재 위치와 밝혀진 곳을 볼 수 있는 지도
   - 세이브 포인트

공격?
 - 기본적인 공격 : 재장전 가능한 투사체
    - 타겟팅
    - 범위에 들어온 몬스터를 타겟팅 가능
    - 타겟팅은 Tab 키로 변경 가능
    - 타겟팅된 몬스터는 빨간색 원으로 표시
    - 타겟팅되지 않은 몬스터는 파란색 원으로 표시
      - 타겟팅 -> 사정거리 안에 들어갔을 때.
      - 몬스터의 
 - 투사체 튕기기
    - 튕긴 투사체는 타겟팅된 몬스터에게

몬스터?
 - idle 상태에서 대기
 - 플레이어가 일정 거리내에 도달하거나, 플레이어에게 피격받을 시
   alert상태에 돌입
 - alert상태에서
   - 일정시간마다 투사체 공격
   - 사정거리 안에 들어오면 이동을 멈춤
   - 사정거리에서 벗어나면 플레이어를 쫓아옴
   - 다시 사정거리 안으로 들어오면 반복
 - 몬스터가 피격당할 시, 깜빡이는 애니메이션

보스?
 - 하나의 방을 따로 둠
 - idle 상태에서 가까이 가면 alert 상태에 돌입
 - 랜덤 패턴 반복.
   - 바닥 레이저
   - 점프 타이밍 맞춰야하는 투사체
